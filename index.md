# Usage


<h3><center>Color</center></h3>

To change the color of your name the commands are:

 - /namecolor
 - /nc


For colors, the commands are used as follows:

 - /namecolor [color identifier]
 -  /nc [color identifier]

To reset your name, run the command(s):

 - /namecolor reset
 - /nc reset
